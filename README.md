# Design-Aware Timing Optimization with OpenROAD and LLM

This project explores design-aware timing optimization using:

- OpenROAD
- Python automation
- QoR report parsing
- Adaptive strategy search
- LLM-guided optimization

The system analyzes timing reports and automatically explores
different optimization strategies for timing closure.

Main optimization parameters:
- clock_period
- CORE_UTILIZATION
- PLACE_DENSITY_LB_ADDON
- TNS_END_PERCENT

The goal is to build a lightweight AI-for-EDA prototype suitable
for research-oriented summer camp interviews.
